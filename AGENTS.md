# Repository Guidelines

## Project Structure

This repository contains a Vite web app in `apps/web` and a FastAPI backend in `backend`.

## Commands

- `pnpm dev`
- `pnpm test`
- `pnpm db:migrate`
- `pnpm init:project --name aged-your-project`

## Naming

- New derived projects must keep the `aged-*` prefix.
- Avoid adding concrete business language to the template baseline.
